CSS:




CSS stands for "Cascading Style Sheets". Simply put, CSS defines how HTML elements are displayed in a webpage. CSS is responsible for the colors, formatting, and overall style/theme of your page. CSS can also control the layout/style of multiple web pages all at once which saves a lot of time and work.

CSS can be used directly within your HTML code, or it can reside in a file all of its own, typically called "styles.css".
